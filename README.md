# 便签助手 (Memo Notes)

轻量级网页便签应用，支持中英文切换与自动保存。所有数据存储在浏览器 `localStorage` 中，无需后端服务。

---

## 功能特性

- **新建便签**：点击「+ 新建便签」或对应英文按钮创建新便签卡片
- **实时编辑**：标题和正文均支持直接输入，每次修改自动保存到浏览器本地存储
- **删除便签**：每条便签右上角带有删除按钮，点击即可移除
- **中英双语**：左上角按钮一键切换界面语言（中文 / English）
- **全屏模式**：右上角按钮进入/退出浏览器全屏，适合专注编辑
- **自动保存**：所有操作即时写入 `localStorage`，刷新页面不丢失

---

## 使用方法

1. 在浏览器中打开 `index.html`
2. 点击下方「+ 新建便签」创建第一条便签
3. 直接点击标题或正文区域开始编辑，内容会自动保存
4. 左上角切换语言，右上角切换全屏

---

## 技术实现

| 技术点 | 说明 |
|--------|------|
| 数据存储 | 浏览器 `localStorage`（键名 `memo_notes_data`） |
| 国际化 | 纯 JavaScript `L` 对象 + `data-key` 属性驱动 |
| XSS 防护 | 自定义 `escapeHtml()` 转义 `<` `>` `&` `"` |
| 全屏 API | 标准 `requestFullscreen` + webkit 前缀兼容 |

## 代码审查已知问题

| # | 问题描述 | 严重程度 |
|---|----------|----------|
| 1 | **全屏按钮未国际化** — 按钮文本 `"全屏"` 硬编码，切换英文后不会更新为 "Fullscreen" | 中 |
| 2 | **删除按钮 title 硬编码** — `title="删除"` 未使用国际化，切换英文时仍显示中文提示 | 低 |
| 3 | **`saveNotes()` 缺少异常处理** — `localStorage.setItem` 在超出配额或隐私模式下会抛出异常，导致保存失败时静默丢失数据 | 中 |
| 4 | **`body` 溢出时可能双滚动条** — `body` 同时设置 `height: 100vh` 与 `overflow-y: auto`，内容较多时可能存在嵌套滚动 | 低 |
